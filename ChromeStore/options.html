Script Caller

This extension provides a customizable icon and menu in Chrome toolbar to run user scripts at browser and/or at local PC.

This extension requires native client that is installed in local Windows PC. (see here)

How to setup menu and user scripts

Menu specification file and script files should be stored in native client "host" folder in local PC as follows.
  • Write specification of icon and menu in customize/browserAction.json file in native client "host" folder.
    JSON key value
    title Title of the icon(menu)
    icon Relative path name of icon image file. It should be 16x16 PNG file.
    menu An array of popup menu items
    menu[].title Title of the menu item
    menu[].matches (optional) An array of URL prefix match strings. The menu item is shown only if one of the string matches the URL of active page.
    menu[].injectionScript (optional) Injection script file name. This file must be stored in ./customize folder in the "host" folder.
    This file contains javascript code that is injected to the active page in the current tab when the menu item is clicked. The returned values from this script are passed to the native script as arguments of main function.
    menu[].nativeScript (optional) Native script file name. This file must be stored in ./customize folder in the "host" folder.
    This file contains javascript code that is executed by Node.js in local PC after executing the injection code. In the code, function main(argument) must be defined.

  • Write injection script and/or native script files for each menu item.
  • See example JSON file and script files in the native client "host" folder.
