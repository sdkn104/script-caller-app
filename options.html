Script Caller

This extension provides a customizable icon and menu in Chrome toolbar to run user scripts at browser and/or local PC.

This extension requires native client to be installed in local Windows PC. (see here)

How to setup menu and user scripts

Menu specification and script files should be stored in native client "host" folder in local PC as follows.
  • Write specification of icon and menu in customize/browserAction.json file in native client "host" folder.
    key value
    title Title of the icon(menu)
    icon Relative path name of icon image file. It should be 16x16 PNG file.
    menu An array of popup menu items
    menu[].id An unique string for ID of the menu item
    menu[].title Title of the menu item
    menu[].matches An array of URL prefix match strings. The menu item is shown if one of the string matches the URL of active page.

  • Write injection code and native code for each menu item. See example code files in the native client "host" folder.
    file name content
    customize/injectionCode_{menu_id}.js javascript code that is injected to the active page on clicking the menu item. The return values are passed to the native code as arguments.
    customize/nagtiveCode_{menu_id}.js javascript code that is executed by Node.js in local PC after executing the injection code. In the code, function main(argument) must be defined.
